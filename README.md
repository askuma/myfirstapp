# Resume Site

Static resume site generated from the provided LinkedIn PDF export.

## Deploy on GitHub Pages
1. Commit and push the repository to GitHub (repo: `myfirstapp`).
2. In GitHub, go to **Settings â†’ Pages**.
3. Under **Build and deployment**, choose **Deploy from a branch**.
4. Select **Branch: main** and **Folder: / (root)**, then **Save**.
5. Your site will be available at `https://<your-username>.github.io/myfirstapp/`.

## Local Preview
Open `index.html` in a browser.
